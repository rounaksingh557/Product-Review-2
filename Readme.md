## Setup
- NOTE: This is setup was used in [Windows Subsystem For Linux](https://learn.microsoft.com/en-us/windows/wsl/about)

```
pip install virtualenv
```

```sh
virtualenv .
```

```sh
source lib/activate
```

```sh
python -r requirements.txt
```

```sh
flask run
```
